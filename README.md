# Customer Behavior Dashboard (Power BI)

## Project Overview
This project analyzes customer purchasing behavior using a cleaned dataset and visualizes key insights through an interactive Power BI dashboard. The dataset was cleaned using Python (Pandas) in Jupyter Notebook and then used to build the dashboard in Power BI.

## Objectives
- Analyze customer purchasing patterns
- Identify top-performing product categories
- Understand customer demographics and spending behavior
- Evaluate subscription distribution among customers
- Present insights through an interactive dashboard

## Tools and Technologies
- Python
- Pandas
- Jupyter Notebook
- Power BI
- Data Visualization

## Data Preprocessing
The dataset was cleaned and transformed using Python before visualization.

Steps performed:
- Removed missing values
- Standardized column names
- Handled inconsistent data
- Created Age Group categories
- Exported cleaned dataset as CSV

## Dashboard Features

### Key Performance Indicators (KPIs)
- Total Number of Customers
- Average Purchase Amount
- Average Review Rating

### Customer Subscription Analysis
- Donut chart showing percentage of subscribed vs non-subscribed customers

### Revenue by Category
- Clothing
- Accessories
- Footwear
- Outerwear

### Sales by Category
- Shows number of purchases for each product category

### Revenue by Age Group
- Young Adult
- Adult
- Middle-aged
- Senior

### Sales by Age Group
- Displays number of purchases made by each age group

## Interactive Filters
- Gender
- Category

## Key Insights
- Footwear category generates the highest revenue
- Senior customers contribute the most revenue
- Around 71% of customers are not subscribed
- Average purchase amount is approximately $59.76
- Average review rating is around 3.75
